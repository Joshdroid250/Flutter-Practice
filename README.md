# Flutter-Practice
# 🚀 Mi Repositorio de Estudio: Dart & Flutter

Este repositorio contiene todo el código, ejercicios y aplicaciones que voy desarrollando a medida que avanzo en el curso **[Flutter - Móvil: De cero a experto](https://www.udemy.com/course/flutter-cero-a-experto/)** de Fernando Herrera en Udemy.

El objetivo es dominar las bases de Dart, comprender el funcionamiento de los widgets, manejar diferentes gestores de estado y aplicar arquitecturas limpias.

---

## 📌 Contenido y Estructura del Curso

El código está organizado siguiendo los bloques principales y proyectos de la guía oficial:

### 🎯 1. Introducción a Dart (Sección 2)
* Conceptos clave del lenguaje: variables, null-safety, funciones, parámetros orientados a objetos.
* Programación Orientada a Objetos: clases, herencia, mixins, clases abstractas.
* Asincronía: `Futures`, `Streams` y `async/await`.

### 📱 2. Primeros Pasos y Aplicaciones
A continuación se listan los proyectos principales que se desarrollan en el curso:

* **📂 `📁 counter_app` (Secciones 4):** Introducción a Flutter, widgets básicos, `StatelessWidget` y `StatefulWidget`.
* **📂 `📁 yes_no_app` (Secciones 5-6):** Chat interactivo, consumo de peticiones HTTP, manejo de enfoques nativos y layouts de mensajes.
* **📂 `📁 toktik_app` (Sección 7):** Clon de videos verticales tipo TikTok. Primer contacto con layouts avanzados y controladores de video.
* **📂 `📁 widgets_app` (Secciones 9-11):** Catálogo de componentes de Material 3, navegación con GoRouter e introducción a la gestión de estado con **Riverpod**.
* **📂 `📁 cinemapedia` (Secciones 12-17):** Aplicación de películas aplicando conceptos de *Clean Architecture* (Datasources y Repositories), manejo avanzado de Riverpod, ShellRoutes y persistencia local.
* **📂 `📁 forms_app` (Secciones 19-20):** Manejo de formularios, validaciones y uso de **BLoC** (FlutterBloc y Cubits).
* **📂 `📁 push_app` (Secciones 21-24):** Integración de Firebase Cloud Messaging, Push Notifications y Local Notifications.
* **📂 `📁 teslo_app` (Secciones 25-30):** Proyecto final full-stack utilizando un backend con autenticación JWT, protección de rutas, manejo de cámaras, galería y carga de archivos.

---

## 🛠️ Tecnologías y Herramientas

* **Lenguaje:** Dart 3.x
* **Framework:** Flutter 3.x
* **Gestores de Estado:** Riverpod / BLoC
* **Navegación:** GoRouter
* **Editor:** VS Code / Android Studio
